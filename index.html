About me

Hi, my name is Stephan Olivier and I have been coding most of my life. I started as a kid on a Commodore 64, got a job when I was 18 as a system administrator and have always learned most of my skills on the job. For the last 10+ years I have been working as developer on a large telco billing system, using the applications own programming language, Perl and shell scripting for the backend quite a lot of SQL.

I already played around building websites for fun in the early ninetees, but I haven't done much with it for the last 10 years, so all this Javascript and Bootstrap stuff is pretty new to me :)

I am currently studying to become a Full Stack Developer using freeCodeCamp and I will use this portfolio to showcase my work.
